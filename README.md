# OpenCart Manual Testing Project

> A complete manual testing project on the OpenCart E-Commerce demo website.  
> Application Under Test: (https://demo.opencart.com)

---

## About This Project

This project is part of my QA portfolio where I perform end-to-end manual
black-box testing on the OpenCart e-commerce platform.  
It covers all major modules from user registration to admin panel testing.

##  Test Approach

- **Testing Type:** Manual Black-Box Testing
- **Testing Level:** System & Integration Testing
- **Techniques:** Equivalence Partitioning, Boundary Value Analysis, Exploratory Testing
- **Browsers:** Chrome (primary), Firefox, Edge
- **Environment:** demo.opencart.com
- **Bug Tracking:** Excel Sheet
